For successful execution, replies will be in the form of {"error": False, "message": "description"}
For unsuccessful execution, replies will be in the form of {"error": True, "message": "description"}

Api Response Code and their meaning:
	200 - Successful
	401 - Unsuccessful
	442 - Incorrect password
	444 - User already exists
	445 - Email ID already exists
	446 - Username should be atleast 8 characters
	447 - Password should be atleast 8 characters
	448 - Bad OTP
	449 - User doesn't exist
	601 - Portfolio doesn't exist
	602 - Portfolio Price doesn't exist
	603 - You currently don't own this stock
	650 - Insufficient funds
	651 - Insufficient quantity of funds to sell
